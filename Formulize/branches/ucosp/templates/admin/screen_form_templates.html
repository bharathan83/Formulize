You can type PHP code into any or all of the three template boxes below. Code in the top template box will replace the default user interface above the form. Code in the elements template box will replace the default way that each form element is displayed. Code in the bottom template box will be rendered below the form.

See the bottom of this page for documentation about how to include buttons and other interface elements in your templates.

Top and Bottom Templates

If you specify any PHP code in the Top or Bottom Templates, it will be used to control the appearance of the space either above or below the form.

To include buttons and controls, use these variables:

Variable Name Description
$title Text containing the form's title.
$formMetaData Text describing which user created the form entry and the date and time at which it was created. Also describes which user last modified the entry and the date and time at which it was last modified. This variable will be empty if the current entry is new.
$formInstructions Text with general instructions about the form.
$saveInstructions Text describing how to save the form.
$doneInstructions Text describing what to do when you are done.
$printBtn A button
$saveBtn A button to validate and save the form.
$allDoneBtn A done button.

Elements Template

You do not need to create a foreach loop or any other loop structure in this template. The PHP code you specify will be executed inside a loop that runs once for each form element.

Variable Name Description
$elementCaption Text containing the element's caption.
$elementDescription Text containing the element's description or help text.
$elementField Section of code containing the element field(s).
$elementIsRequired A boolean that is set to true if this element is required or false otherwise.
